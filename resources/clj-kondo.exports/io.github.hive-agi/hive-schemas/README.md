# hive-schemas clj-kondo export

This export teaches consumers about the named test vars generated by
`deftrifecta-from-schema`, `deftrifecta-predicate`, and
`deftriad-from-schema`, plus the proof-oriented `deftrifecta-*` wrappers.

It is loaded automatically by `clj-kondo --copy-configs` when
`io.github.hive-agi/hive-schemas` is present on the dependency classpath.
